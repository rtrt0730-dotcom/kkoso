# 김비서 — 업무 대시보드

개인 업무 관리 및 데이터 분석을 위한 통합 대시보드 플랫폼입니다.

## 🎯 주요 기능

- **📅 할 일 관리** — 우선순위별 정렬, 체크박스로 완료 체크, 실시간 삭제
- **⏰ 주간 일정** — 월~금 요일별 일정 표시, 긴급 항목 강조
- **💰 매출 현황** — 제품별·지역별·날짜별 차트, 상세 테이블
- **📊 시각화** — 막대/도넛/꺾은선 차트, 진행률 바
- **🌍 세계 시간** — 서울·뉴욕 실시간 시계, DST 자동 반영
- **🌤️ 날씨 정보** — 각 도시 기온·상태 실시간 표시
- **📱 반응형 디자인** — iPhone 15 Pro 최적화

## 📁 파일 구조

```
워크숍-실습/
├── index.html                          # 메뉴 랜딩 페이지
├── CLAUDE.md                           # 프로젝트 문서
└── 2-체험용-샘플/
    ├── 업무대시보드.html               # 메인 대시보드 (할 일, 일정, 매출)
    ├── 매출보고서.html                 # 매출 분석 상세 보고서
    ├── subtrac-분석보고서.html         # SubTrackr 서비스 분석
    ├── mbis-분석보고서.html            # MBIS 조명 브랜드 분석
    ├── 마케팅업무흐름.svg              # 마케팅 5단계 흐름도
    ├── 매출데이터.csv                  # 원본 매출 데이터
    ├── 회의록.txt                      # 마케팅팀 주간회의 기록
    └── 행사포스터.jpg                  # AI 인프라 포럼 포스터
```

## 🛠️ 기술 스택

- **프론트엔드**: HTML5, CSS3 (반응형 디자인)
- **인터랙션**: Vanilla JavaScript (프레임워크 미사용)
- **차트**: Chart.js (제품별 매출, 지역별 분석)
- **API**: wttr.in (실시간 날씨 데이터, 인증 불필요)
- **배포**: Vercel (GitHub 연동 자동 배포)
- **버전 관리**: Git + GitHub

## 🎨 핵심 특징

- **다크 모드** — 업무대시보드는 다크 테마로 야간 사용 최적화
- **라이트 모드** — 분석 보고서는 밝은 테마로 인쇄 친화적
- **모바일 우선** — 480px 이하에서 레이아웃 최적화
- **실시간 업데이트** — 시계는 1초, 날씨는 10분마다 갱신
- **데이터 기반** — CSV 원본 데이터 기반 차트 시각화
